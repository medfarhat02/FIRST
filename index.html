HTML Basics

  • HTML: HyperText Markup Language
  • Tags: Used to structure content
  • Attributes: Provide additional information about elements

Basic HTML Tags

Some basic HTML tags include:

  • <p>: Paragraph
  • <a>: Anchor (link)
  • <img>: Image
  • <ul>: Unordered List
  • <ol>: Ordered List
  • <li>: List Item

Feel free to explore and experiment with HTML!
